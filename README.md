# Sunshine_Discord-Bot

  Um bot para Discord com múltiplas funcionalidades, desenvolvido com foco em aprendizado e evolução contínua.
  O projeto tem como objetivo crescer futuramente para algo além de um simples bot, se tornando uma aplicação mais completa.

--Funcionalidades:
    -Comandos básicos de interação
    -Sistema de embeds (mensagens estilizadas)
    -Respostas automáticas
    -Organização de comandos (prefix e/ou slash commands)

--Tecnologias utilizadas
    -JavaScript
    -Node.js
    -Biblioteca para Discord (ex: discord.js)

--Como executar o projeto
    1- Clone o repositório:
      git clone https://github.com/seu-usuario/Qin_Discord-Bot.git;
    2- Acesse a pasta do projeto:
      cd Qin_Discord-Bot
    3- Instale as dependências:
      npm install
    4- Configure o token do bot:
      Crie um arquivo .env ou configure diretamente no código
      Adicione o token do seu bot do Discord
    5- Execute o bot:
      node index.js

--Configuração
  Antes de rodar o projeto, você precisa:
    -Criar um bot no Discord Developer Portal
    -Copiar o token do bot
    -Convidar o bot para um servidor

--Status do projeto
    Em desenvolvimento
    O projeto está em constante evolução, com novas funcionalidades sendo adicionadas.

--Objetivo
  Este projeto foi criado com o objetivo de:
    -Praticar programação
    -Aprender mais sobre bots e automação
    -Evoluir para um sistema mais robusto no futuro
